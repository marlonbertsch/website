## Overview
Built during my Spring 2026 semester, this project demonstrates retrieval-augmented generation (RAG) on course PDFs.

## Tech Stack
- **Backend**: Python, FastAPI, LangChain
- **Frontend**: React + Tailwind
- **Deployment**: Docker, Railway

## Key Features
- Chat with uploaded lecture slides
- Quiz generation based on weak areas
- Offline mode with local models
